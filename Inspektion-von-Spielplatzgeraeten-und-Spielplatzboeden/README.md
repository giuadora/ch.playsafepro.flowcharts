# Inspektion von Spielplatzgeräten und Spielplatzböden

## Über dieses Paket

Dieses Paket enthält zwei Ablaufdiagramme für die normkonforme Inspektion von Spielplätzen gemäss **SN EN 1176-7:2020** (Anleitung für Installation, Inspektion, Wartung und Betrieb):

- **Neuer Spielplatz**: Zeigt den Inbetriebnahme-Prozess vom Bauabschluss über die Bauabnahme und Inspektion nach Installation bis zur Spielplatzeröffnung und Erstellung des Inspektionsplans.
- **Bestehender Spielplatz**: Zeigt den laufenden Inspektionszyklus mit den drei Inspektionstypen (Visuelle Routine-Inspektion, Operative Inspektion, Jährliche Hauptinspektion) sowie Berichtswesen und Instandhaltungs-Management.

**Zielgruppe:** Spielplatzbetreiber, Gemeinden, Immobilienverwaltungen und Eigentümer, die ihre Sorgfaltspflicht nach Art. 58 OR (Werkeigentümerhaftung) erfüllen müssen.

**Version 2.0** ist optimiert für professionellen Druck im Format DIN A4 hochkant.

## Bearbeitung mit draw.io

### Datei öffnen

**Option 1: Online (keine Installation erforderlich)**

1. Öffnen Sie https://app.diagrams.net im Browser
2. Klicken Sie auf „Open Existing Diagram"
3. Wählen Sie die gewünschte Datei (`Neuer-Spielplatz.drawio` oder `Bestehender-Spielplatz.drawio`) aus
4. Das Diagramm wird geladen und kann sofort bearbeitet werden

**Option 2: Desktop-App (empfohlen für regelmässige Nutzung)**

1. Laden Sie draw.io Desktop herunter: https://github.com/jgraph/drawio-desktop/releases
2. Installieren Sie die App für Ihr Betriebssystem (Windows, macOS, Linux)
3. Öffnen Sie die gewünschte drawio-Datei direkt mit der Desktop-App

### Diagramm bearbeiten

**Text ändern:**
- Doppelklicken Sie auf einen Knoten
- Bearbeiten Sie den Text direkt im Knoten
- Drücken Sie Enter oder klicken Sie ausserhalb, um die Änderung zu speichern

**Knoten verschieben:**
- Klicken und ziehen Sie einen Knoten an eine neue Position
- Verbindungslinien passen sich automatisch an (bei aktivierter orthogonaler Linienführung)

**Farben anpassen:**
1. Wählen Sie einen Knoten aus (einfacher Klick)
2. Im rechten Panel finden Sie „Fill" (Füllung) und „Stroke" (Rahmen)
3. Klicken Sie auf die Farbfelder, um neue Farben auszuwählen
4. Verwenden Sie die Hex-Werte unten für Konsistenz mit dem aktuellen Farbschema

**Neue Knoten hinzufügen:**
1. Ziehen Sie eine Form aus der linken Bibliothek auf die Zeichenfläche
2. Das Diagramm verwendet diese Formen:
   - **Rechteck** = Prozessschritt / Aktion
   - **Raute** = Entscheidungspunkt
   - **Abgerundetes Rechteck** = Start / Ende
   - **Parallelogramm** = Dokument / Bericht
   - **Doppelt umrandetes Rechteck** = Unterprozess (vordefinierter Prozess)

**Firmenlogo einfügen:**
1. Wählen Sie den Platzhalter-Bereich oben rechts aus (falls vorhanden)
2. Menü: Arrange → Insert → Image
3. Wählen Sie Ihr Firmenlogo (PNG, JPG oder SVG)
4. Positionieren und skalieren Sie das Logo nach Bedarf

### Export

**Als PDF exportieren:**
1. Menü: File → Export as → PDF
2. Wählen Sie „Selection only" falls Sie nur einen Teil exportieren möchten
3. Klicken Sie auf „Export"

**Als PNG exportieren:**
1. Menü: File → Export as → PNG
2. Wählen Sie die Auflösung:
   - 300 DPI für hochwertigen Druck
   - 100 DPI für Web und Präsentationen
3. Klicken Sie auf „Export"

**Als SVG exportieren:**
1. Menü: File → Export as → SVG
2. Aktivieren Sie „Include a copy of my diagram" falls Sie die Bearbeitungsinformationen beibehalten möchten
3. Klicken Sie auf „Export"

## Import in Microsoft Visio

### Direkt-Import (empfohlene Methode)

Microsoft Visio kann draw.io-Dateien direkt öffnen:

1. Öffnen Sie Microsoft Visio
2. Menü: Datei → Öffnen
3. Wählen Sie im Dateityp-Filter „Alle Dateien (*.*)"
4. Navigieren Sie zur gewünschten drawio-Datei
5. Klicken Sie auf „Öffnen"

Der Import funktioniert ab Visio 2013.

### Alternative: Import über SVG

Falls der Direkt-Import nicht funktioniert:

1. Öffnen Sie die gewünschte drawio-Datei in draw.io
2. Exportieren Sie als SVG (siehe oben)
3. Öffnen Sie die SVG-Datei in Visio: Datei → Öffnen
4. Visio konvertiert das SVG automatisch in bearbeitbare Shapes

**Hinweis:** Nach dem SVG-Import sind einige Elemente möglicherweise gruppiert. Verwenden Sie „Gruppierung aufheben" um einzelne Shapes zu bearbeiten.

## Farbschema

Das Diagramm verwendet ein print-optimiertes Farbschema mit klarem Schwarzweiss-Kontrast:

| Farbe | Verwendung | Füllfarbe (Hex) | Rahmen (Hex) |
|-------|------------|-----------------|--------------|
| **Blau** | Neuer Spielplatz | #D6E4F0 | #2B579A |
| **Grün** | Inspektionen | #DFF0D8 | #3C763D |
| **Gelb** | Mängel / Massnahmen | #FFF3CD | #856404 |
| **Grau** | Start / Ende | #E9ECEF | #6C757D |

Diese Farben gewährleisten gute Lesbarkeit sowohl auf dem Bildschirm als auch im Schwarz-Weiss-Ausdruck.

## Technische Hinweise

### SVG-Datei

Die Datei `Spielplatzinspektionen.svg` ist eine Vektorgrafik, die in jedem modernen Browser geöffnet werden kann:

- **Im Browser öffnen:** Doppelklicken Sie auf die SVG-Datei
- **In Dokumente einfügen:** Drag & Drop in Word, PowerPoint, InDesign, etc.
- **Drucken:** SVG ist verlustfrei skalierbar — ideal für Poster und Ausdrucke in beliebiger Grösse

**Einschränkung:** SVG-Dateien sind nicht direkt editierbar. Für Änderungen verwenden Sie bitte die draw.io-Datei und exportieren Sie anschliessend neu als SVG.

### Mermaid-Dateien (für technische Nutzer)

Die Dateien `Neuer-Spielplatz.md` und `Bestehender-Spielplatz.md` enthalten den Quellcode der beiden Diagramme im Mermaid-Format. Mermaid ist eine Markup-Sprache für Diagramme, die in vielen technischen Dokumentationen verwendet wird.

**Verwendungsmöglichkeiten:**
- Einbetten in Markdown-Dokumentationen (GitHub, GitLab, Confluence, Notion)
- Rendern mit Mermaid CLI:
  - `mmdc -i Neuer-Spielplatz.md -o Neuer-Spielplatz-mermaid.svg`
  - `mmdc -i Bestehender-Spielplatz.md -o Bestehender-Spielplatz-mermaid.svg`
- Live-Editor für Anpassungen: https://mermaid.live
- Einbetten in Websites mit mermaid.js

**Hinweis:** Die Mermaid-Dateien sind vor allem für technische Nutzer gedacht, die die Diagramme in ihre Dokumentation einbetten oder programmatisch verarbeiten möchten. Für normale visuelle Bearbeitungen verwenden Sie bitte die draw.io-Dateien.
